# STM32F446RE Peripheral Driver Development



## Project Overview
This repository contains bare-metal driver development for the STM32F446RE.

### Projects
- GPIO Blink
- USART Button
- SPI Communication
- I2C Temperature Example

## Driver Analysis Report
The report is available in the `DRIVER_ANALYSIS_REPORT` folder.
